# About this repository

Hello, humans! I'm Dk4LL, a [Flamengo](https://www.flamengo.com.br/) fan, freelance programmer, and lover [Unix-like system](https://www.linfo.org/unix-like.html), specifically [OpenBSD](https://www.openbsd.org/) and [Arch Linux](https://archlinux.org/). This repository houses my personal [NeoVim](https://neovim.io/) configuration, if you want to clone and use my configuration, **guse** (go and use)!

## Basic tools
Please install on your system:

* [NeoVim](https://neovim.io/) (this one is obvious).
* [tree-sitter](https://tree-sitter.github.io/tree-sitter/).
* [ripgrep](https://github.com/BurntSushi/ripgrep).

## Languages and Tools
It is also necessary to install the following languages and tools on your system:

* [astro](https://www.npmjs.com/package/astro)
* [clang](https://clang.llvm.org/)
* [typescript](https://www.typescriptlang.org/)
* [ghcup](https://www.haskell.org/ghcup/)
* [opam](https://opam.ocaml.org/)
* [live-server](https://www.npmjs.com/package/live-server)
* [tailwindcss](https://tailwindcss.com/)
* [swi-prolog](https://https://www.swi-prolog.org/)
* [lean](https://leanprover-community.github.io/get_started.html)

## Language servers
Also install on your system:

* [astro-ls](https://www.npmjs.com/package/@astrojs/language-server)
* [texlab](https://github.com/latex-lsp/texlab)
* [vscode-langservers-extracted](https://github.com/hrsh7th/vscode-langservers-extracted)
* [pyright](https://github.com/microsoft/pyright)
* [lua-language-server](https://github.com/LuaLS/lua-language-server)
* [clangd](https://github.com/clangd/clangd)
* [typescript-language-server](https://www.npmjs.com/package/typescript-language-server)
* [Prolog Language server](https://github.com/jamesnvc/lsp_server)
* [@tailwindcss/language-server](https://www.npmjs.com/package/@tailwindcss/language-server)
* [Prolog Language server](https://github.com/jamesnvc/lsp_server)

You should use ghcup to install HLS, Cabal and Stack. Furthermore, use opam to install ocamllsp and use Mason to install jdtls.
